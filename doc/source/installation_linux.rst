Installation on Linux
=====================

Python 3.x
----------
.. code-block:: bash

	sudo apt-get install -y freeglut3 freeglut3-dev binutils-gold
	sudo pip3 install arcade

Python 2.x
----------
.. code-block:: bash

	sudo apt-get install -y freeglut3 freeglut3-dev binutils-gold
	sudo pip install arcade
